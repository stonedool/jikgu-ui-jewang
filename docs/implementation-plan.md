# 구현 계획

## Agent 역할

1. 페이지 이해 Agent
   - 현재 브라우저 페이지와 선택 텍스트에 근거해 답변합니다.
   - 상품 자체, 옵션, 리뷰, 배송 문구, 페이지 문장 번역을 담당합니다.
   - 법령/통관/관세 판단은 직구지원 RAG Agent로 넘깁니다.

2. 직구지원 RAG Agent
   - 직구 관련 법령, 공식 안내, 소비자 보호 문서, 배송대행 체크리스트를 검색합니다.
   - 답변에는 출처와 문서 날짜를 포함합니다.
   - 최신성이나 적용 가능성이 불명확하면 확인 필요로 표시합니다.

3. 외국어 퀴즈 Agent
   - 페이지에 보이는 단어와 표현을 후보로 삼습니다.
   - 사용자 수준에 맞는 단어를 골라 객관식/문맥형 퀴즈를 만듭니다.
   - 정답/오답 기록은 확장 프로그램 로컬 저장소에 보관합니다.

## 라우팅 기준

- 페이지/상품/옵션/리뷰/번역: 페이지 이해 Agent
- 관세/통관/수입 제한/개인통관고유부호/배송대행/소비자 분쟁: 직구지원 RAG Agent
- 단어/표현/퀴즈/뜻/빈칸: 외국어 퀴즈 Agent
- "이 상품 직구해도 괜찮아?" 같은 혼합 질문: 페이지 정보 추출 후 RAG 검색

## 개인정보 처리

- 확장 프로그램은 입력값을 수집하지 않고 보이는 텍스트만 수집합니다.
- 서버와 확장 프로그램 모두 이메일, 전화번호, 카드번호로 보이는 문자열을 마스킹합니다.
- 페이지 전체 저장은 하지 않고 요청 처리 중 컨텍스트로만 사용합니다.
